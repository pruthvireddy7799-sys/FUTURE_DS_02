# FUTURE_DS_02
# 📊 Customer Churn Analysis Dashboard

## 🎯 Objective

The goal of this project is to analyze customer churn behavior and identify key factors influencing why customers leave a service.

---

## 📊 Dashboard Features

* Total Customers (KPI)
* Churned Customers (KPI)
* Churn Rate (KPI)
* Churn Distribution (Yes vs No)
* Churn by Contract Type
* Churn by Monthly Charges (binned)
* Churn by Tenure (binned)

---

## 💡 Key Insights

* Customers with **month-to-month contracts** have the highest churn rate
* Customers with **higher monthly charges** are more likely to churn
* **New customers (low tenure)** are more likely to leave
* Long-term customers show better retention
